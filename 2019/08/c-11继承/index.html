C++11继承

八月 27, 2019 发布在 技术小白, C++

　　我们都知道C++面向对象(OOP)一共有四个特性，抽象、多态、继承和封装，在此，通过自己的学习和记录，本篇文章将简要的介绍一下继承的相关概念，方便与理解。

继承是什么

　　何为继承？简单来说，继承就是在一个已有类的基础上构建一个新的类，这个新构建的类我们就称之为子类或者派生类，而原有的类就是父类或者基类。基类就是比较抽象的，派生类呢就是比较具体的。派生类和基类“是一种”关系，就比如说有这么一个基类：形状，那么它的派生类就可以有圆形，矩形等等。又或者说定义一个基类：车，那么它的派生类就可以有自行车，汽车，公交车等等。

继承的意义

　　当我们知道了什么是继承了之后，我们不禁又要提问，那我们为什么要使用继承呢？这就好比我们在编写一个程序时，往往要使用很多的类，有时候这些类都是类似的，如果我们使用了继承的话，就可以提高代码重用率，缩短程序开发时间时间，可以说大大提高的编程的效率，对程序员来说还是非常友好的。

继承的实例

　　说了这么多，所以我们该如何使用继承呢？用这里将举一个小小的例子，比如有一个基类Shape，和一个派生类Circle，那么继承的话就可以写成： class Circle:public Shape{};其中public表示为公有继承，当然，也会有私有(private)继承和保护(protect)继承。而且，当我们继承基类时，我们不能直接访问基类的不可访问(private)变量，我们需要调用基类的构造函数，借此访问基类的私有变量。既然如此，如果我们不希望类被继承的话我们该怎么办？C++11引入了final特殊标识符，可以使得类不能被继承，例如：class Shape final{}；这个时候如果使用class Circle: public Shape{}来继承Shape类的话，程序就会报错。

代码实现

　　现在通过一个代码实现简单展示一下继承。首先是头文件基类Shape类：

#include<iostream>
#include<string>
#include<array>
using std::string;
using std::array;
using namespace std::string_literals;

enum class Color {
	white, black, red, green, blue, yellow,
};//关键字enum，定义一个枚举类型
class Shape {
private:
	Color color{ Color::black };//就地初始化
public:
	Shape() = default;//无参构造函数
	Shape(Color color_) {
		color = color_;
	}
	Color getColor() {
		return color;
	}
	void setColor(Color color_) {
		color = color_;
	}
	string toString() {
		array <string, 6>c{ "white"s,"black"s,"red"s,"green"s,"blue"s,"yellow"s };//使用数组类array
		return c[static_cast<int>(color)];//static_cast就是强制类型转换color为整型值
	}
};

　　然后是继承类Circle和主函数：

#include<iostream>
#include<string>
#include"Shape.h"
#include"Circle.h"
//类Circle继承基类Shape
class Circle : public Shape {
public:
	Circle();//无参构造函数
	Circle(Color color_) :Shape(color_) {

	}
};
//主函数
int main() {
	Shape s1{ Color::blue };
	Circle c1{ Color::green };

	std::cout <<"Shape:"<< s1.toString() << std::endl;
	std::cout <<"Circle:"<< c1.toString() << std::endl;
}

　　最后得到的输出结果为：

Shape：blue
Circle: green

构造函数继承

　　说完了类继承，在C++11中派生类还能继承基类的构造函数，当然析构函数和友元函数不能继承。当我们要继承基类的构造函数时，我们可以使用using A: A这种形式继承所有基类的构造函数，不能仅继承某个构造函数。下面简单展示代码实例：

#include<iostream>
using std::cout;
using std::endl;
class A {
public:
	A() {
		cout << "A is:()" << endl;
	}
	A(int i) {
		cout << "A is:" << i << endl;
	}
};
class B : public A {
public:
	using A::A;//继承构造函数
	B() {
		cout << "B is:()" << endl;
	}
	B(double x): A(static_cast<int>(x)) {
		cout << "B is:" << x << endl;
	}
};
int main() {
	B b1;
	B b{ 2.2 };

}

　　运行结果：

A is:()
B is:()
A is:2
B is:2.2

　　通过运行结果我们可以看出主函数通过对象调用类B的构造函数，而调用类B的构造函数之前会先调用基类的构造函数。
