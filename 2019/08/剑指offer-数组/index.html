题目一

数组中重复的数字

　　在一个长度为n的数组里的所有数字都在0到n-1的范围内。 数组中某些数字是重复的，但不知道有几个数字是重复的。也不知道每个数字重复几次。请找出数组中任意一个重复的数字。 例如，如果输入长度为7的数组{2,3,1,0,2,5,3}，那么对应的输出是第一个重复的数字2。

思路理解

　　从头到尾扫描数组，假设此时处在下标为i的位置，如果下标i和下标对应的数字，假设为m相等，则i往后移一位，如果不相等，那么比较下标i所对应数字m，和下标m所对应数字n，如果m=n,说明重复，如果不相等，则交换下标i和下标m所对应的值，然后继续比较i,直到找到重复为止。

代码实现

class Solution {
public:
	bool duplicate(int numbers[], int length, int* duplication) {
		if (numbers == 0 || length == 0) {
			return false;//如果数组长度为空或者长度为零则直接返回false
		}
		for (int i = 0; i < length; i++) {
			while (numbers[i] != i) {//当下标i不等于对应值进入循环
				if (numbers[i] == numbers[numbers[i]]) {
					*duplication = numbers[i];//找到重复的数字
					return true;
				}
				else {//如果不相等，则交换两下标的值
					int temp = numbers[i];
					numbers[i] = numbers[temp];
					numbers[temp] = temp;
				}
			}
		}
		return false;
	}
};

题目二

二维数组中的查找

　　在一个二维数组中（每个一维数组的长度相同），每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序。请完成一个函数，输入这样的一个二维数组和一个整数，判断数组中是否含有该整数。

思路理解

　　因为二维数组从左到右以及从上到下都是递增，所以取数组中右上角的数，如果右上角的数比该数小，那就说明右上角左边的数都比该数小，那么这一行可以剔除。如果右上角的数比该数大，那就说明右上角下面的数都比该数大，呢么这一列可以剔除。依次比较，知道找到该数为止。

代码实现

class Solution {
public:
	bool Find(int target, vector<vector<int>>array) {//定义整数和二维数组
		int rows = array.size();//初始化行
		if (rows < 0) {
			return false;
		}
		int cols = array[0].size();//初始化列
		if (cols < 0) {
			return false;
		}
		int row = 0, col = cols - 1;//取得右上角的数
		while (row < rows && col >= 0) {
			if (target > array[row][col]) {
				row++;//如果右上角的数比该整数小，则行++
			}
			else if(target < array[row][col]) {
				col--;//如果右上角的数比该整数大，则列--
			}
			else {
				return true;
			}
		}
		return false;
	}
};
